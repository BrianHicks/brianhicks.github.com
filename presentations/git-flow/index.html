GIT FLOW

you're gonna love this

What is it?

A collection of Git extensions to provide high-level repository operations for Vincent Driessen's branching model.

— Git flow project page

What's in it for me?

  • Deployments and rollbacks
  • Release management
  • Standard branching model across projects

Installation

  • Windows: installer
  • Mac OS X: available in brew, macports (or manual installation)
  • Linux: package for Debian/Ubuntu; not needed on servers
Start a new branch 
git flow feature start add-more-bacon
Work in the branch 
git ci -m "adding more bacon"
Finish the feature branch 
git flow feature finish add-more-bacon
Start a new release 
git flow release start 1.0.0
Bump version number 
git ci -m "bump version number to 1.0.0"
Finish release 
git flow release finish 1.0.0
Start a new hotfix 
git flow hotfix start 1.0.1
Fix the problem 
git ci -m "fixed problem with flux capacitor"
Bump version number 
git ci -m "bump version number to 1.0.1"
Finish hotfix 
git flow hotfix finish 1.0.1
